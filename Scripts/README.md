# Scripts

## setup.sh

Устанавливает все зависимости проекта одним запуском:

1. **SPM зависимости** — разрешает и скачивает Swift Package Manager пакеты (SwiftLint и др.)
2. **Viper шаблон** — копирует шаблон Viper модуля в Xcode Templates

```bash
./setup.sh
```

## install_template.sh

Устанавливает только Viper шаблон в Xcode.

```bash
./install_template.sh
```
